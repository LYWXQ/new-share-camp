# 代码重构

## 功能
- 分析代码结构，识别可优化的部分
- 提供代码重构建议，包括：
  - 函数提取与模块化
  - 变量命名优化
  - 代码重复消除
  - 性能瓶颈识别与优化
  - 代码风格统一
- 提供重构前后的对比分析

## 使用场景
- 当代码变得难以维护或理解时
- 当需要提升代码性能时
- 当代码重复率较高时
- 当需要统一代码风格时
- 当准备进行大规模功能迭代前

## 工作流程
1. 分析现有代码结构
2. 识别重构机会
3. 制定重构计划
4. 执行重构操作
5. 验证重构结果
6. 提供重构报告

## 注意事项
- 重构前应确保代码有足够的测试覆盖
- 重构过程中应保持功能不变
- 重构后应运行完整的测试套件
- 对于大型项目，建议分阶段进行重构
