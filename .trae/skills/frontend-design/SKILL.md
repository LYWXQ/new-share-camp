本技能指导创建具有独特风格、避免通用"AI 垃圾"美学的生产级前端界面。实现具有卓越审美细节和创意选择的真实可用代码。

用户提供前端需求：要构建的组件、页面、应用程序或界面。他们可能包括关于用途、受众或技术约束的上下文信息。

## 设计思维

在编码之前，先理解上下文并确定一个**大胆**的美学方向：
- **目的**：这个界面解决什么问题？谁使用它？
- **风格**：选择一个极端方向：极简主义、极繁主义混乱、复古未来主义、有机/自然、奢华/精致、 playful/玩具感、编辑/杂志风、粗野主义/原始、装饰艺术/几何、柔和/ pastel、工业/实用等。有如此多的风格可供选择。将这些作为灵感，但设计一个符合美学方向的风格。
- **约束**：技术要求（框架、性能、可访问性）。
- **差异化**：什么让这令人**难忘**？什么是人们会记住的那一件事？

**关键**：选择一个清晰的概念方向并精准执行。大胆极繁主义和精致极简主义都有效——关键是意图性，而非强度。

然后实现可用的代码（HTML/CSS/JS、React、Vue 等），具备以下特点：
- 生产级且功能完整
- 视觉震撼且令人难忘
- 具有清晰美学观点的连贯性
- 每个细节都精心打磨

## 前端美学指南

重点关注：
- **排版**：选择美观、独特、有趣的字体。避免使用 Arial 和 Inter 等通用字体；选择能够提升前端美学的独特字体；出人意料、有个性的字体选择。将独特的展示字体与精致的正文字体搭配。
- **色彩与主题**：坚持统一的美学。使用 CSS 变量保持一致性。主色调搭配锐利点缀色胜过胆怯、均匀分布的调色板。
- **动效**：使用动画实现效果和微交互。HTML 优先使用纯 CSS 解决方案。React 可用时使用 Motion 库。专注于高影响力时刻：一个精心编排的页面加载配合交错显示（animation-delay）比分散的微交互创造更多愉悦感。使用滚动触发和令人惊喜的悬停状态。
- **空间构图**：出人意料的布局。不对称。重叠。对角线流动。打破网格的元素。充裕的负空间**或**受控的密度。
- **背景与视觉细节**：创造氛围和深度，而不是默认使用纯色。添加符合整体美学的上下文效果和纹理。应用创意形式，如渐变网格、噪点纹理、几何图案、分层透明度、戏剧性阴影、装饰性边框、自定义光标和颗粒叠加。

永远不要使用通用的 AI 生成美学，如过度使用的字体家族（Inter、Roboto、Arial、系统字体）、陈词滥调的色彩方案（特别是白色背景上的紫色渐变）、可预测的布局和组件模式，以及缺乏上下文特定特征的千篇一律设计。

创造性地诠释并做出令人意外的选择，让人感觉是真正为上下文设计的。没有设计应该相同。在明暗主题、不同字体、不同美学之间变化。永远不要在不同生成中趋同于常见选择（例如 Space Grotesk）。

**重要**：将实现复杂度与美学愿景相匹配。极繁主义设计需要包含大量动画和效果的复杂代码。极简主义或精致设计需要克制、精准，并仔细关注间距、排版和微妙细节。优雅来自于很好地执行愿景。

记住：Claude 能够创造非凡的创意作品。不要退缩，展示当跳出思维定式并完全投入独特愿景时，真正能创造出什么。
