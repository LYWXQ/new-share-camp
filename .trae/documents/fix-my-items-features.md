# 修复我的发布页面功能

## 问题描述
上次对话中有两个功能未正确实现：

### 1. 物品详情页 - 本人发布标识
- 本人发布的物品不应该显示"联系发布者"和"立即借用"按钮
- 在发布者栏需要显示"本人发布"标识

### 2. 批量操作 - 查看详情按钮
- 仅在批量操作模式下，给物品卡片添加"查看详情"按钮
- 点击按钮可以跳转到物品详情页

## 实现方案

### 功能1：物品详情页本人发布标识

#### 修改文件
`d:\work\share-camp-master\frontend\src\pages\item-detail\item-detail.vue`

#### 实现步骤
1. 检查是否已添加 `isOwnItem` 计算属性
2. 检查发布者栏是否显示"本人发布"标识（蓝色徽章）
3. 检查是否已隐藏"联系发布者"和"立即借用"按钮
4. 检查样式是否正确

#### 预期效果
- 本人发布的物品：显示蓝色"本人发布"徽章，隐藏底部操作按钮
- 他人发布的物品：显示绿色"已认证"徽章（如果已认证），显示底部操作按钮

### 功能2：批量操作查看详情按钮

#### 修改文件
`d:\work\share-camp-master\frontend\src\pages\my-items\my-items.vue`

#### 实现步骤
1. 在批量操作模式下（`v-if="isBatchMode"`），在物品卡片底部添加"查看详情"按钮
2. 按钮样式使用蓝色（与编辑按钮一致）
3. 点击按钮调用 `goToDetail(item.id)` 跳转到详情页
4. 使用 `@click.stop` 阻止事件冒泡

#### 预期效果
- 批量模式下每个物品卡片显示"查看详情"按钮
- 点击按钮可以正常跳转到物品详情页
- 不影响批量选择功能
